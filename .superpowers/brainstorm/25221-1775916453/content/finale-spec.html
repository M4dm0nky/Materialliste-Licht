Katalog Redesign — Spec

Finale Architektur · 5 Welten · Flexible Tiefe · Gruppen-Sektionen

Die 5 Welten

🔌 Datenwelt 3 EBENEN Hersteller-Untergruppen
⚡ Stromwelt 2 EBENEN direkt Artikel
💡 Lichtwelt 3 EBENEN Hersteller-Untergruppen
🔩 Riggingwelt 2 EBENEN direkt Artikel
📦 Verbrauchswelt 2 EBENEN direkt Artikel

Visuelles Konzept

Kabel (lengths)
Wie bisher: Typ = Sektion, Längen als Zeilen
Geräte (qty)
Gruppe = Sektion, jedes Gerät = eine Zeile. Kein eigener Sektionskopf pro Gerät.
3-Ebenen Welten
Untergruppen-Zeile (z.B. Hersteller) zwischen Gruppe und Artikel
Unterschied sichtbar?
Ja — Kabel haben Längen-Spalte, Geräte nur #/Spare/ImProjekt/Diff

Datenmodell

Katalog-Typ — neue Felder

"MAC Viper XIP": {
  cat: "Lichtwelt",
  group: "Moving Lights", ← schon vorhanden
  subgroup: "MARTIN", ← NEU (optional)
  unit_type: "qty",
  items: [{n:"MAC Viper XIP"}]
}
"DMX 5-Pin": {
  cat: "Datenwelt",
  group: "DMX",
  ← kein subgroup bei Kabeln
  unit_type: "lengths",
  items: [{n:"DMX", l:"1m"}, ...]
}

Gruppen — Erweiterung

← world-Feld NEU auf Gruppen-Objekt
{id:"moving-lights", name:"Moving Lights", world:"Lichtwelt"}
