# Materialliste Licht

Touring Production Material Management — Web-App konvertiert aus `LichtMaterialliste_V3.xltm`.

## Features

- Alle Excel-Logiken: `DIFF = (# Kabel + Spare) − Im Projekt` → **Grün / Rot**
- 4 Kategorien, 30+ Sektionen, alle Originaleinträge
- LocalStorage-Persistenz — Daten bleiben gespeichert
- CSV-Export (Excel-kompatibel mit BOM)
- Kollabierbare Sektionen, globaler Status-Indikator

## Öffnen

`index.html` direkt im Browser öffnen — kein Server nötig.

## GitHub Pages

Settings → Pages → Branch: main → `https://<user>.github.io/materialliste-licht/`
